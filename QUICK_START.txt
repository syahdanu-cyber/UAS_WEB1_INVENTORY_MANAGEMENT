QUICK START GUIDE - SISTEM MANAJEMEN INVENTORY
==============================================================================

LANGKAH CEPAT UNTUK MEMULAI:

1. PERSIAPAN DATABASE
   - Buka phpMyAdmin atau MySQL client
   - Buat database baru: CREATE DATABASE manajemen_inventory;
   - Import file database.sql

2. KONFIGURASI
   - Edit file: config/database.php
   - Sesuaikan username dan password MySQL Anda

3. AKSES APLIKASI
   - Buka browser: http://localhost/manajemen_inventory/
   - Login dengan:
     * Username: admin
     * Password: admin123

4. FILE YANG PERLU DIBUAT
   Lihat IMPLEMENTATION_GUIDE.md untuk panduan lengkap membuat file:
   - barang/barang.php (List & Read)
   - barang/tambah.php (Create)
   - barang/edit.php (Update)
   - barang/hapus.php (Delete)
   - supplier/* (sama seperti barang)
   - transaksi/masuk.php
   - transaksi/keluar.php
   - laporan/laporan.php
   - laporan/export_pdf.php
   - laporan/export_excel.php

5. INSTALL LIBRARY (untuk Export PDF & Excel)
   composer require tecnickcom/tcpdf
   composer require phpoffice/phpspreadsheet
